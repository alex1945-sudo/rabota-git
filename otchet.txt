# Отчет по практике

## Работа с GitHub

1. Создал репозиторий 'rabota-git' на GitHub
2. Связал локальный репозиторий с удаленным:
   git remote add origin https://github.com/alex1945-sudo/rabota-git.git
3. Отправил код на GitHub: git push -u origin main
4. Успешная отправка подтверждена сообщением в терминале

## Публикация сайта

1. Настроил GitHub Pages в настройках репозитория
2. Сайт доступен по адресу: https://alex1945-sudo.github.io/rabota-git/

## Вывод

Освоил базовые операции Git: создание репозитория, коммиты, отправку кода на GitHub. Настроил хостинг статического сайта через GitHub Pages.
